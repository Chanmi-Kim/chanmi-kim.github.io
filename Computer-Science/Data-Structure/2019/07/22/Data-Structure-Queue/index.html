Data Structure : Queue(큐)

  • 큐는 컴퓨터의 기본적인 자료 구조의 한가지로, 먼저 집어 넣은 데이터가 먼저 나오는 FIFO (First In First Out)구조로 저장하는 형식을 말합니다.

예시문제

  • 주어진 N(2<= N <=100)개의 수를 순서대로 Queue에 넣은 후 하나씩 꺼내 화면에 출력하시오.
입력 출력
2 // 테스트 케이스 수
5 // 데이터 크기
1 2 3 4 5
5
5 4 2 3 1		 #1 1 2 3 4 5
#2 5 4 2 3 1
